## [XMRTH](https://xmrth.cc/auth/register?code=kkDm) （2023）
- [x] 1000~3000G直连
- [x] 50G~300G中继
- [x] 5 ~ 16 元/每月

直连结点流量很多，某些结点偶尔会抽风一下，建议配合Clash，按照规则进行自动切换。
5元的有10G中继流量。
10元的增加了几个中继结点，中继流量50G。
16元的有200G中继流量。
